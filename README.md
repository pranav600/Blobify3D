# Blobify3D

Blobify3D is a modern React/Next.js application designed to allow creators to explore, design, and share beautiful 3D shapes. It boasts a visually stunning interface with micro-animations, glassmorphism, and a high-end aesthetic tailored for a premium user experience.

## Features
- **Dynamic 3D Shapes Library**: Browse endless unique blobs and 3D geometries.
- **Premium Design Aesthetic**: Highly refined UI with a focus on typography, color palettes, and polished interactions.
- **Modern Tech Stack**: Built with Next.js, Framer Motion for smooth animations, and Tailwind CSS for rapid and beautiful styling.
- **Responsive Layout**: Seamlessly functioning on both desktop and mobile devices.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- Frontend: Next.js (React), Typescript
- Styling: Tailwind CSS
- Animations: Framer Motion
- Assets: Lucide React for consistent iconography and DiceBear for avatars

## Deployment
This project is set up and optimized to be deployed on platforms like Vercel.
